Vault on AWS

Client identity and environment details are protected under a Non-Disclosure Agreement (NDA).
As part of the clients internal service catalog, the modularization of a HashiCorp Vault instance on AWS was essential. To achieve this, a Terraform/Jenkins CI/CD pipeline was established, enabling the dynamic creation of HashiCorp Vault clusters with an Active-Active architecture upon repository push in AWS. This streamlined the rapid deployment of a Vault instance in response to requests from application teams.
